Web-based client for [Yarr!](https://github.com/bluecap-se/yarr)


## Install

```Bash
$ git clone https://github.com/bluecap-se/yarr.client.git
$ cd yarr.client
$ make install
```


## Usage

```Bash
$ make serve

Server started on localhost:9090
```


## Run tests

```Bash
$ make test
```
